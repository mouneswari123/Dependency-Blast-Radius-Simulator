## Development Approach

The project was built incrementally:

1. Designed database schema using Prisma and PostgreSQL.
2. Implemented backend REST APIs using Node.js and Express.
3. Built frontend using React and Next.js.
4. Integrated frontend and backend using Axios.
5. Implemented dependency graph visualization using React Flow.
6. Added failure simulation and blast radius calculation using DFS.
7. Added circular dependency detection.
8. Implemented simulation history and dashboard analytics.
9. Added search, filtering, and validation features.
10. Performed testing and bug fixes.
